# Play 심사용 계정 제공 템플릿

아래 내용을 Play Console > 앱 콘텐츠 > 앱 액세스에 입력하세요.

## 기본 설명
- 본 앱은 로그인 후 사용 가능합니다.
- 공개 회원가입은 제공하지 않습니다.
- 아래 심사용 계정으로 전체 주요 기능 접근이 가능합니다.

## 심사용 계정 정보
- 계정 ID: [REVIEWER_ID]
- 비밀번호: [REVIEWER_PASSWORD]
- 계정 권한: [GENERAL 또는 ADMIN]

## 접근 경로
- 앱 실행 -> 로그인 화면 -> 계정/비밀번호 입력 -> 로그인

## 검토 포인트
- 일반 기능 확인: 출석, 리그, 회비 조회/참여
- 관리자 기능 확인(권한 계정 제공 시): 회원 관리, 회비 관리

## 추가 안내
- 로그인 실패 시 연락 채널: [운영 이메일]
- 계정 만료/잠김 대비 예비 계정
  - 예비 계정 ID: [REVIEWER_BACKUP_ID]
  - 예비 비밀번호: [REVIEWER_BACKUP_PASSWORD]
