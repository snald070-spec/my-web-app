# Play Console 입력본 (한국어, 2026-04-11)

이 문서는 Play Console에 그대로 붙여넣어 사용할 수 있는 최종 입력용 문안입니다.

## 1) 앱 액세스 (App access)
다음 문안을 Play Console > 앱 콘텐츠 > 앱 액세스에 입력하세요.

- 본 앱은 로그인 후에만 사용 가능합니다.
- 일반 사용자 공개 회원가입은 제공하지 않습니다.
- 심사용 테스트 계정은 Play Console의 앱 액세스 항목에 별도 제공하겠습니다.
- 로그인 경로: 앱 실행 -> 로그인 화면 -> 계정/비밀번호 입력
- 로그인 후 접근 범위:
  - 일반 회원: 출석, 리그, 회비 상태 조회/참여
  - 관리자/마스터: 회원 관리, 회비 관리, 운영 로그

## 2) 데이터 세이프티 (요약 입력 가이드)
- 데이터 수집: 예
- 데이터 공유: 아니오(현재 릴리즈 구성 기준)
- 전송 중 암호화: 예
- 데이터 삭제 요청 수단: 필요(개인정보처리방침에 이메일/폼 명시)

## 3) 개인정보처리방침 핵심 문구 초안
아래 항목을 개인정보처리방침 본문에 포함하세요.

- 수집 항목
  - 계정/프로필: 사번(또는 계정 ID), 이름, 권한(role), 부서/본부, 이메일(선택)
  - 서비스 운영: 출석 응답, 리그 참여/결과, 회비 납부 상태/이력
  - 운영 보조(관리자 기능): 입금자명, 입금액, 입금시각, 매칭 상태
- 이용 목적
  - 로그인 인증 및 권한 제어
  - 출석/리그/회비 기능 제공
  - 운영 감사 로그 및 보안 모니터링
- 제3자 제공/공유
  - 현재 릴리즈 구성에서 광고/분석 목적의 활성 데이터 공유 흐름은 확인되지 않음
- 보관 및 파기
  - 서비스 운영 목적 범위에서 보관 후 내부 정책에 따라 파기
- 이용자 권리
  - 정정/삭제/처리정지 요청 접수 채널 제공(이메일 또는 웹 폼)

## 4) 삭제 요청(필수 보강 문구)
현재 구조는 관리자 주도 계정 관리 방식입니다.

- 서버에는 관리자 삭제 엔드포인트가 존재합니다.
- 다만 사용자 셀프 삭제 UI/플로우는 현재 앱에서 확인되지 않았습니다.
- 따라서 Play 정책 대응을 위해 아래 중 최소 1개를 즉시 운영하세요.
  - 개인정보처리방침에 삭제 요청 이메일 명시
  - 삭제 요청 웹 폼 URL 제공

권장 문구:
"회원은 개인정보 삭제를 요청할 수 있으며, 요청은 [운영 이메일/폼]으로 접수됩니다. 본인 확인 후 지체 없이 처리 결과를 안내합니다."

## 5) 제출 전 최종 체크
- 개인정보처리방침 공개 URL 입력
- 데이터 세이프티 문항 제출
- 앱 액세스 테스트 계정 제공
- 내부 테스트 트랙 업로드 및 테스터 지정
