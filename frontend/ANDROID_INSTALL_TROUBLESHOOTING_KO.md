# Android 설치 트러블슈팅 (테스터용)

## 1) 설치 버튼이 비활성화됨
- 원인: 보안 설정에서 외부 설치가 차단됨
- 조치: 설정 > 보안(또는 앱) > 알 수 없는 앱 설치 허용 활성화

## 2) "앱이 설치되지 않았습니다" 메시지
- 기존 동일 앱 충돌 가능
- 조치:
  1. 기존 앱 삭제
  2. APK 재다운로드
  3. 저장 공간 확보 후 재설치

## 3) 설치 후 실행 즉시 종료
- 조치:
  1. 앱 완전 종료 후 재실행
  2. 기기 재부팅
  3. 동일 증상 시 기기명/OS 버전/발생 시점 공유

## 4) 로그인 실패
- 계정 정보 재확인
- 네트워크 연결 확인
- 계속 실패 시 운영자에게 계정 잠금/비밀번호 확인 요청

## 5) 공유 시 필수 정보
- 기기명, OS 버전, 오류 시각, 재현 순서, 스크린샷
